# Messenger

## Команды

- **npm run start**
  Сборка и запуск на порту 3000
- **npm run build**
  Сборка
- **npm run test**
  Запуск тестов
- **npm run lint**
  Запуск линтера
- **npm run lint:fix**
  Исправление ошибок линтера
